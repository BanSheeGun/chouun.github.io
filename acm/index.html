浙江大学城市学院ACM-ICPC竞赛简章

By ChouUn

发表于 2015-09-12

导语

很荣幸大家能抽出宝贵的时间，阅读鄙人的这个页面。
计算系的大部分新生是零基础，这是情理之中的。而ACM竞赛给了你一个公正的平台，你能和985/211高校的学生一样，从同一条起跑线出发。只要肯认真学习，就能在亚洲区域赛上，与他们同台竞技。
多说无益，上干货。

以下我将从简介、优势、联络、赛制等几方面，进行一个较为详细的介绍，缺乏耐心的童鞋可以直接看问答。

简介

关于ACM-ICPC

ACM (Association for Computing Machinery)，即美国计算机协会，是一个国际计算机界具有悠久历史的权威性组织。
ICPC (International Collegiate Programming Contest)，即国际大学生程序设计竞赛。
ACM-ICPC（以下简称「ACM竞赛」）是由美国计算机协会主办的一项旨在展示大学生创新能力、团队精神和在压力下编写程序、分析和解决问题能力的年度竞赛。ACM竞赛始于1970年，于1996年由上海大学引入中国大陆，目前已经发展为国际上 历史最悠久、 规模最庞大、 最具影响力 的大学生计算机竞赛，并被冠以「程序设计的奥林匹克」的尊称。
作为计算机系的学生，ACM是展现你计算机才华的广阔舞台，也是学习成果的直接体现，更是与 高端信息企业 直接对话的最好机会。

关于ACM实验室

程序设计竞赛工作室 (Programming Contest Studio)，位于北校区 - 理科四号楼 - 二楼 - 225。
「程序设计竞赛工作室」是计算机应用创新实践基地内的学科竞赛实验室之一，始建于2007年4月。
实验室工作与「程序设计」类教学课程相结合，以「ACM竞赛」为主要训练内容与途径，因此也称作「ACM实验室」。
ACM实验室旨在培养学生以程序设计水平和能力为代表的综合专业素质，以提高学科竞赛水平，并促进相关教学工作。

优势

就业

按zucc历年数据来看，我们分院就业转正后，平均工资 约四五千；而ACM实验室中，有获奖经历的优秀学长，工资平均在 一万以上。
近年来中国大陆赛区的比赛得到了 IBM、 华为技术有限公司、 上海欢乐互娱网络科技有限公司 和 北京爱奇艺公司 是等企业的大力支持。其中IBM是ACM国际大学生程序设计竞赛全球总决赛的独家赞助商，同时也是地区级比赛的主要赞助商，已经连续19年赞助此项赛事。其他赞助过的企业有 阿里巴巴集团、 谷歌、 微软、 百度、 腾讯、 Sun中国工程研究院、 联想、 金山软件、 中航文化、 中国移动、 达内集团 等。
这些国内外著名企业都是认可ACM竞赛的，你参赛的经历能为实践积累经验，你获得的奖项能为就业铺平道路。

资源

我们有专属的教师团队进行答疑解惑。有英俊风趣的张老师，有瘦削较真的柯老板，还有和蔼可亲的高燕姐。他们不仅关注你的学业成绩，也很关心你的健康状况与校园生活。
我们也有专门的实验室提供训练场地。学校对我们有一定的资源倾斜，理四225是正式成员的集训场所，而理四221则是编外成员的训练场所。后者能让你在别人晚自习的时候，有更多提高代码能力的机会，在理论与实践的结合中升华。
我们更有专业的学长们进行交流指导。高年级学长带低年级学弟，是ACM实验室的一贯作风。我们将所学的知识薪火相传，没有对初学者的歧视，只有亦师亦友的爱护；甚至在军训期间，也会有学长跨越校区，送予你烈日下的一杯清凉。

校队

我校于2004年开始正式参与ACM国际大学生程序设计竞赛，历年预选赛征战的足迹包括上海、北京、长春、沈阳、哈尔滨、广东、四川、西安、天津、吉林、南阳、合肥等，而且计划内的经费完全由学校报销，无须各人承担。
参赛十余年，我校代表队培养了一批又一批的计算机科学领域的优秀人才。无论是在科研还是工作岗位上，他们都显示出了惊人的创造力和强大的竞争力。他们中的有的人，目前正在国外著名学府继续学习深造；有的人，在大型跨国IT公司从事高端科技产品的研发与管理工作。

联络

竞赛

第一学期，会举行分院级别的「程序设计课程竞赛」，参与者限定大一，个人赛。此时你已经会C语言基本的语法了，因此以思维题为主，考验代码能力和逻辑能力。
第二学期，会举行城院级别的校赛，参与者限定大一大二，个人赛。此时会适度的考算法和数据结构。此中表现优秀的人，会组队参加浙江大学的校赛、以及浙江省的省赛。
第三学期，你就有机会组队参加亚洲级别的区域赛了，能代表学校与各大高校同台竞技，然后争金夺银、大放异彩。而且值得注意的是，到了这一步，奖项都是亚洲级别的。 奖学金、二课分、红卡、大公司的offer，指日可待。

问答

Q：我该如何加入实验室
A：我们主要从 第一学期和第二学期的两次选拔赛 中选择满足招新要求成员。大部分同学需要通过这个途径，获得实验室的招收名额。如果有特殊需要，可以参见特殊申请。

Q：第一次选拔赛在甚么时候
A：第一学期的「程序设计课程竞赛」暨「ACM实验室新生选拔赛」，一般在期中考之后举办。我们会选择排名较前的部分同学，优先入驻实验室熟悉环境。

Q：招收名额是固定的么？人员是否会变动？
A：我们会根据「实际空位」和「参赛成绩」决定名额。选拔赛代表了单次的发挥，并不意味着你坐稳了一席之地，懒惰或发挥失常可能会使你失去名额。

Q：我应该为选拔赛做何准备
A：首要六个字，「学好程序设计」。这不但是你的专业课，也是这项竞赛的基础。熟悉C语言可以参考浙大版《C语言程序设计（第3版）》题目集，会了基本语法以后可以考虑刷城院入门水题和杭电入门水题。网络上还有其他Online Judge可以刷题，详见后面某个QA。

Q：选拔赛的题型怎么安排的
A：历年新生赛可以在这里查到，提交程序需要到这里寻找对应的题目。

Q：女生是不是很难进实验室啊
A：在公平竞赛的前提下，女生的条件会适当放宽，因此只要努力的方向对，还是不难进实验室的。

Q：ACM跟其他社团、组织冲突么？
A：并不冲突，我们实验室有很多排球队、足球队、甚至是马球队的，也有学长是二课的负责人，或者是各种社团的。实在要说冲突的，就是活动时间偶尔会重合。据不完全统计，实验室的脱单率达到了惊人的50%。

Q：我该如何联络你们，在哪刷题，用什么刷题
A：理四221机房在开学一定时间后开放，可以干与程序相关的事，请勿做PPT、打游戏。有专业人员负责签到，有疑问可以趁机提问。

Q：提交以后的返回结果看不懂啊
A：程序完成之后提交运行，运行的结果会判定为AC(正确)、WA（错误）、TLE（超时）、MLE（超出内存限制）、RE（运行错误）、PE（格式错误）中的一种。

Q：学校有没有开设相关课程
A：

  1. 第一学期有「C语言程序设计」必修课，唔，夯实基础吧。
  2. 第二学期有「数据结构」必修课，同上。
  3. 第二学期周二晚上有「ACM基础」限选课，相当于计算和软工的专业选修课，其他专业的公选课。
  4. 第二学期假期中会有「ACM短学期」能替代普通的短学期，你能学到别人学不到的新姿势。
  5. 第三学期周二晚上有「ACM进阶」公选课。

申请

招新要求

我们对新成员的要求：

  1. 人品过硬，不弄虚作假，要尊重老师；性格要好，和其他成员能融洽相处。
  2. 愿意无条件服从集训队安排的训练、组队、参赛以及其他统一的活动。
  3. 意志坚定，切忌一日曝、十日寒。愿意长期付出时间，而不是三心二意。
  4. 「c语言程序设计」和「数据结构」挂科者，不予考虑。在两次院赛中成绩优异者、或有效题量高者，优先考虑。
  5. 没有第五点了

如果你觉得能做到以上几点，可以考虑与我们并肩作战。

特殊申请

我们主要从 第一学期和第二学期的两次选拔赛 中选择成员，也有其他途径加入：

  1. 高年级同学仍然有意愿加入实验室，可以提交入队申请，并单独安排选拔比赛。
  2. 数学、信息学竞赛省一等奖及以上，可以提交入队申请，并单独安排选拔比赛。
  3. 在BestCoder和CodeForces中的Rating，如果三场或以上超过1500分的2015级新生，可以直接提交入队申请。

场地

ACM实验室位于理四225，即红星所在。

门口有比较明显的标志，方便大家辨别。

赛制

流程明细

  1. 参赛队伍最多由三名参赛队员组成；
  2. 竞赛中至少命题6题，至多命题13题，试题描述为英文，比赛时间为5个小时；
  3. 竞赛可以使用的语言：C++、C、Java和Pascal。但final赛只有C/C++；
  4. 重点考察选手的算法和程序设计能力，不考察任何Windows编程知识；
  5. 选手可携带任何非电子类资料，包括书籍和打印出来的程序等；
  6. 评委负责将结果（正确或出错的类型）通过网络尽快返回给选手，除此之外不提供任何额外帮助。

区域赛和总决赛

ACM 国际大学生程序设计竞赛由各大洲区域赛(Regional Contests)和全球总决赛(World Finals)两个阶段组成。
各大洲区域赛一般安排在每年的9-12月举行，每个区域第一名自动获得参加全球总决赛的资格；全球总决赛安排在第二年的上半年举行。
每所大学可以有多支队伍参加区域赛，但只能有 一支 队伍参加全球总决赛。

后记

最后我想说，竞赛毕竟是竞赛，ACM/ICPC竞赛和其他竞赛是一样的，到最后无论你用什么去美化，去修饰，如果你的目的是拿奖，那必然是残酷的。所以要端正自己的态度，自律才是最重要的。我很欣赏watashi的10000小时理论，不拿出花10000小时做好一件事的精神，就是不够努力，就不应该奢望变强，更加不好意思说遗憾。
最后的最后，如果一开始的目的不是竞赛，不是拿奖，还是不要参与了，大学可以有别的方法过的很精彩。

以上文字来自知乎。虽然我引用了这段话，但是我并不是完全赞同的。赛场是敌人，赛后是挚友，ACM竞赛就是这样一种微妙的东西。大家彼此之间经常交流，建立起了深厚的友谊，团队的荣誉感也随之催生。学长们就算退役多年，已久会关注自己学校队伍的战况，甚至回来看看小学弟，没有什么年级的隔膜，这是其他组织很难做到的一点。

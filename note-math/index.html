最大公約數 (Greatest Common Divisor)

歐幾里德算法

https://en.wikipedia.org/wiki/Euclidean_algorithm

歐幾里德算法 (Euclidean Algorithm)：
$ \begin{eqnarray} gcd(a,b) = \begin{cases}
a, & b = 0 \cr
gcd(b, a \bmod b), & otherwise
\end{cases} \end{eqnarray} $

證:
令 $ r = gcd(a,b) $
則 $ r \mid a, r \mid b $
故 $ r \mid (a - b) $
引 $ r \mid (a - kb) \Rightarrow r \mid (a - \lfloor \frac{a}{b} \rfloor b) \Rightarrow r \mid (a \bmod b) $
故 $ gcd(a,b) \equiv gcd(b, a \bmod b) $

莫比烏斯函數

https://en.wikipedia.org/wiki/M%C3%B6bius_function

莫比烏斯函數 (Möbius Function)：
一般用 $ mu(n) $ 或者 $ \mu(n) $ 表示，是個積性函數 (multiplicative function)。
$ \begin{eqnarray}
\mu(n) =
\begin{cases}
x = 1, & n = 1 \cr
x = (-1)^k, & \text{if } n \text{ is the product of } k \text{ different primes} \cr
x = 0, & otherwise
\end{cases}
\end{eqnarray} $

中國剩餘定理

https://en.wikipedia.org/wiki/Chinese_remainder_theorem

中國剩餘定理 (Chinese Remainder Theorem)：
已知 $ M = \prod m_{i} $，$ m_{i} \text{ is a prime number} $，$ S \equiv a_{i} (\bmod m_{i}) $，求 $ S $
先看 $ S \equiv a_{i} (\bmod m_{i}) $ 可以轉化爲 $ S + m_{i}y = a_{i} $，這個公式仍然不利於求解
聯想到 $ ax + by = gcd(a, b) $ 的形式，可以用 擴展歐幾里德 搞一搞
不妨令 $ M_{i} = \frac{ M }{ m_{i} } $，構造 $ M_{i}x + m_{i}y = gcd(M_{i}, m_{i}) = 1 $
這樣搞的好處都有啥？ 誰說對了csy就給他（非金坷拉
化成 $ M_{i}x \equiv 1 (\bmod m_{i}) $，$ \forall j \neq i, M_{i}x \equiv 0 (\bmod m_{j}) $
顯然 $ x $ 是 $ M_{i} $ 關於 $ m_{i} $ 的逆元，發現並不一定要用 $ exgcd $，我們記爲 $ x_{i} $
那麼 $ a_{i}M_{i}x_{i} \equiv a_{i} (\bmod m_{i}) $，$ S = \sum a_{i}M_{i}x_{i} (\bmod M)$

遞推

a[n] = a[n-1]^2 - 2

$ a_{n} = a_{n-1} ^ 2 - 2 $
令 $ a_{0} = x + \frac{1}{x} $
則 $ a_{n} = x ^ {2^n} + \frac{1}{x^{2^n}} $

a[n] = p⋅a[n-1] + q⋅a[n-2]

$ a_{n} = p⋅a_{n-1} + q⋅a_{n-2} $
特徵方程 $ x ^ 2 = px + q $，求得 $ \alpha \beta $。
若 $ \alpha \neq \beta $，則 $ a_{n} = c_{1} \alpha ^ n + c_{2} \beta ^ n $；
若 $ \alpha = \beta $，則 $ a_{n} = (c_{1} + n c_{2}) \alpha ^ n $。
求得 $ c_{1}, c_{2} $，求得 $ p, q $，即求得 $ a_{n} $。

(a+√b)^n

$ S_{n} = \lceil (a + \sqrt{b}) ^ {n} \rceil $
構造 $ (a + \sqrt{b}) ^ {n} + (a - \sqrt{b}) ^ {n} $
$ \because 0 < a - \sqrt{b} < 1 $，
且二項式展開、合併同類項後，只可能有 $ \sqrt{b} ^ {2i} = b ^ i $
$ \therefore S_{n} = (a + \sqrt{b}) ^ {n} + (a - \sqrt{b}) ^ {n} $
$ S_{n} ⋅ ((a + \sqrt{b}) + (a - \sqrt{b})) $
$ =
(a + \sqrt{b}) ^ {n+1} +
(a - \sqrt{b}) ^ {n+1} +
(a + \sqrt{b}) (a - \sqrt{b}) ((a + \sqrt{b}) ^ {n-1} + (a - \sqrt{b}) ^ {n-1}) $
$ = S_{n+1} + (a ^ 2 - b) S_{n-1} $
移項得 $ S_{n+1} = 2a S_{n} + (b - a ^ 2) S_{n-1} $

也可以直接構造上一節 “a[n] = p⋅a[n-1] + q⋅a[n-2]” 的形式，
特徵根 $ a - \sqrt{b}, a + \sqrt{b} $，$ p = 2a, q = b - a ^ 2 $
輕鬆得 $ S_{n+1} = 2a S_{n} + (b - a ^ 2) S_{n-1} $

牛頓迭代法

https://en.wikipedia.org/wiki/Newton%27s_method

對 $ \sqrt{S} $ 構造 $ f(x) = x ^ 2 - S = 0 $
有 $ x_{n+1}
= x_{n} - \frac{ f(x_{n}) }{ f’(x_{n}) }
= x_{n} - \frac{ x_{n}^{2} - S }{ 2x_{n} }
= \frac{1}{2} (x_{n} + \frac{S}{x_{n}}) $

数论

最大公约数(Greatest Common Divisor)

某几个整数共有约数中最大的一个。

欧几里得算法(Euclidean Algorithm)

欧几里德算法：$\begin{eqnarray} gcd(a,b) = \begin{cases}
a, & b = 0 \cr
gcd(b, a \bmod b), & otherwise
\end{cases} \end{eqnarray}$

证明:
令$r = gcd(a,b)$
则$r \mid a, r \mid b$
故$r \mid (a - b)$
引$r \mid (a - kb) \Rightarrow r \mid (a - \lfloor \frac{a}{b} \rfloor b) \Rightarrow r \mid (a \bmod b)$
故$gcd(a,b) \equiv gcd(b, a \bmod b)$

扩展欧几里得算法(Extended Euclidean Algorithm)

莫比乌斯函数(Möbius Function)

一般用$mu(n)$或者$\mu(n)$表示，是个积性函数(multiplicative function)
$\begin{eqnarray} \mu(n) = \begin{cases}
x = 1, & n = 1 \cr
x = (-1)^k, & \text{if } n \text{ is the product of } k \text{ different primes} \cr
x = 0, & otherwise
\end{cases} \end{eqnarray}$

递推

a[n] = a[n-1]^2 - 2

$a_{n} = a_{n-1}^2 - 2$
令 $a_{0} = x + \frac{1}{x}$
则 $a_{n} = x^{2^n} + \frac{1}{x^{2^n}}$

a[n] = p⋅a[n-1] + q⋅a[n-2]

$a_{n} = p a_{n-1} + q a_{n-2}$
特征方程 $x ^ 2 = px + q$，求得 $\alpha \beta$。
若 $\alpha \neq \beta$，则 $a_{n} = c_{1} \alpha ^ n + c_{2} \beta ^ n$；
若 $\alpha = \beta$，则 $a_{n} = (c_{1} + n c_{2}) \alpha ^ n$。
求得 $c_{1}, c_{2}$，求得 $p, q$，即求得 $a_{n}$。

(a+√b)^n

$S_{n} = \lceil (a + \sqrt{b})^{n} \rceil $
构造 $(a + \sqrt{b})^{n} + (a - \sqrt{b})^{n}$
$\because 0 < a - \sqrt{b} < 1$，
且二项式展开、合并同类项后，只可能有 $\sqrt{b}^{2i} = b^i$
$\therefore S_{n} = (a + \sqrt{b})^{n} + (a - \sqrt{b})^{n}$
$S_{n} ((a + \sqrt{b}) + (a - \sqrt{b}))$
$ = (a + \sqrt{b})^{n+1} + (a - \sqrt{b})^{n+1} + (a + \sqrt{b})(a - \sqrt{b})((a + \sqrt{b})^{n-1} + (a - \sqrt{b})^{n-1})$
$ = S_{n+1} + (a ^ 2 - b) S_{n-1}$
移项得 $S_{n+1} = 2a S_{n} + (b - a ^ 2) S_{n-1}$

也可以直接构造上一节“a[n] = p⋅a[n-1] + q⋅a[n-2]”的形式，
特征根 $a - \sqrt{b}, a + \sqrt{b}$，$p = 2a, q = b - a ^ 2$
轻松得 $S_{n+1} = 2a S_{n} + (b - a ^ 2) S_{n-1}$

牛頓迭代法

對於 $\sqrt{S}$，構造 $f(x) = x ^ 2 - S = 0$
有 $x_{n+1} = x_{n} - \frac{ f(x_{n}) }{ f’(x_{n}) } = x_{n} - \frac{ x_{n}^{2} - S }{ 2x_{n} } = \frac{1}{2} (x_{n} + \frac{S}{x_{n}})$
